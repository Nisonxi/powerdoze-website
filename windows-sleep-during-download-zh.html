Windows 電源修正

如何防止 Windows 在下載或傳輸檔案時自動睡眠？

Windows 以鍵盤滑鼠的閒置時間決定是否睡眠，不看網路活動。一個跑了幾小時的種子下載，在 Windows 眼中和空機沒有區別——閒置時間到了就睡。有三種解法：暫時修改睡眠設定、使用防睡工具，或把下載管理員加入應用程式白名單，下載完成後自動恢復正常。

Windows 為什麼會在下載途中睡眠？

原因很簡單：Windows 的閒置計時器只計算鍵盤和滑鼠動作。檔案傳輸、種子下載、轉檔工作和背景備份全都不算「活動」。閒置時間到了（通常 15 到 30 分鐘），Windows 就直接進入睡眠，不管背景有什麼在跑。唯一的例外是應用程式主動向 Windows 申請「電源喚醒鎖」（如 Netflix），但大多數下載工具不會這麼做。

方法一：暫時修改睡眠設定

開啟「設定 → 系統 → 電源與睡眠」，將「插電時，電腦在以下時間後進入睡眠」設為「從不」。開始下載。下載完成後回去把設定改回原來的數值。

適合：偶爾下載且人在旁邊的情況。風險：忘記改回的話，電腦從此不會自動睡眠，每晚持續耗電發熱。

方法二：使用防睡工具（Caffeine、PowerToys Awake、Don't Sleep）

安裝 Caffeine 或啟用 PowerToys Awake，點一下工具列圖示開啟防睡模式，下載完成後再點一次關閉。這類工具的原理是每隔幾秒注入假按鍵或向 Windows 申請電源喚醒鎖，讓閒置計時器不過期。

適合：偶爾使用且習慣記得關閉的人。風險：和方法一一樣，忘記關閉就變成永不睡眠。防睡工具完整比較 →

方法三：PowerDoze 應用程式白名單（自動重設）

安裝 PowerDoze（免費）。進入「應用程式白名單」，加入你的下載管理員（qBittorrent、uTorrent、Transmission、IDM 等）。白名單應用程式執行期間，PowerDoze 阻止睡眠；應用程式關閉的瞬間，正常的睡眠排程自動恢復，不需要任何手動操作。

同樣的方式也適用於轉檔工作（Blender、Premiere、DaVinci Resolve）、備份工具（Backblaze、Robocopy 腳本），或任何不希望被中斷的長時間背景程序。

適合：定期跑下載、轉檔或備份、不想每次手動切換的人。唯一真正的「設定一次、永久有效」解法。

常見問題

為什麼 Windows 在下載途中會自動睡眠？

Windows 只計算鍵盤和滑鼠動作的閒置時間，不計算網路傳輸活動。下載、種子傳輸、檔案複製和轉檔工作都不會重置閒置計時器，逾時後就自動進入睡眠。

不裝程式有辦法阻止睡眠嗎？

有——進入「設定 → 系統 → 電源與睡眠」，把睡眠時間設為「從不」。下載完成後記得改回，否則電腦就此不再自動睡眠。

下載完成後電腦還會自動睡眠嗎？

用 PowerDoze 應用程式白名單的話會——PowerDoze 只在白名單應用程式執行期間阻止睡眠，關閉下載管理員後睡眠排程立即恢復。

這個方法也能防止螢幕關閉嗎？

螢幕關閉和系統睡眠是不同的 Windows 設定。應用程式白名單主要封鎖系統睡眠。如需同時防止螢幕關閉，可以在 PowerDoze 或 Windows 電源設定中另行調整。

PowerDoze 相容 qBittorrent、uTorrent、IDM 等下載工具嗎？

相容——任何 Windows 應用程式都可以加入白名單。PowerDoze 以執行檔名稱偵測程序，可搭配所有下載管理員、備份工具和轉檔程式使用。
