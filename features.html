Scheduled sleep rules

Create time-based rules that control when your PC goes to sleep. Each rule defines a time window and a sleep timeout. PowerDoze evaluates rules every minute and applies the matching one.

  • Set different timeouts for work hours, evening, and late night
  • Filter by day of week (e.g., weekdays only, weekends only)
  • Link a Power Mode to each rule for full automation
  • Rules are evaluated by priority — first match wins
  • Free: up to 3 rules / Pro: unlimited

One-click power modes

Power Modes bundle multiple system settings into a single profile you can switch with one click. Each mode controls:

  • Windows power scheme (Balanced, High Performance, Power Saver, or custom GUID)
  • CPU min/max percentage and boost mode
  • Cooling policy (active vs. passive)
  • Screen timeout (AC and DC separately)
  • PCIe ASPM link state power management
  • NVIDIA GPU power limit (if NVIDIA driver is available)
  • Free: 2 modes / Pro: unlimited

App whitelist

Prevent your PC from sleeping when specific apps are running. Perfect for downloads, rendering, long-running tasks, or any process that shouldn't be interrupted.

  • Organize apps into groups (e.g., "Games", "Work", "Downloads")
  • Scan running processes or installed apps to add quickly
  • Drag-and-drop to reorder between groups
  • Icons auto-extracted from executables

Fullscreen & game detection

PowerDoze automatically detects when a fullscreen application (game, movie, presentation) is running and suspends all sleep rules. When you exit fullscreen, the normal schedule resumes.

  • Works with any DirectX/OpenGL/Vulkan game
  • Also detects fullscreen video players and presentation apps
  • No configuration needed — fully automatic
  • FPS counter available in the hardware monitor overlay

Away mode

When you stop using your PC for a configurable number of minutes, PowerDoze automatically switches to an eco-friendly power mode. When you return, it restores your previous settings.

  • Free: Enable/disable away mode with a fixed idle timeout
  • Pro: Choose which power mode to switch to when away
  • Pro: Separate timeout for AC and battery power
  • Uses Windows idle detection (keyboard + mouse activity)
  • Won't trigger during fullscreen apps or meetings

App foreground auto-switch

Automatically switch to a specific power mode when a particular app is in the foreground. When you switch away from that app, the previous mode is restored.

  • Map any process to a power mode (e.g., Premiere Pro → Performance)
  • Meeting app detection: Teams, Zoom, Slack, Discord, and more
  • Customizable meeting app list
  • Higher priority than schedule rules — foreground always wins

Wi-Fi geofencing

Automatically switch power modes based on which Wi-Fi network you're connected to. Acts as a simple location-aware trigger without GPS.

  • Map Wi-Fi SSIDs to power modes (e.g., "Office-5G" → Balanced)
  • Automatically detects current SSID
  • Useful for "at work" vs. "at home" vs. "at coffee shop" profiles
  • Checks periodically in the background

Battery smart policy

Define automatic power mode switches based on battery level and charging state. Keep Performance when plugged in, switch to Power Saver below 30%.

  • Set battery level thresholds (e.g., below 20% → Power Saver)
  • Separate modes for AC power vs. battery
  • Transition notifications so you know when modes change
  • Works alongside schedule rules and away mode
