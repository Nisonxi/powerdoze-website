FAQ

Frequently asked questions

Is PowerDoze free?
Yes! Core features are free forever. Pro unlocks advanced automation like app-based switching, Wi-Fi geofencing, and analytics. It's a one-time purchase, not a subscription.
Does it send my data anywhere?
No. PowerDoze runs entirely on your local machine. The only network requests are an optional update check and Pro license verification when you sign in. No telemetry, no analytics, no cloud sync.
Why does Windows SmartScreen show a warning?
Code signing certificates cost US$200+/year. As an indie developer, we'll add signing once the project is sustainable. Click "More info" then "Run anyway" to proceed safely.
Can I transfer my license to a new PC?
Yes. Each Pro license works on up to 3 devices. Manage your bound devices from the account page — unbind an old PC to free a slot (one unbind every 30 days).
Does it work on laptops and desktops?
Both. Laptops benefit from battery policies and away mode. Desktops benefit from scheduled sleep rules and app-based power mode switching.
What happens if I don't like it?
The free version has no time limit. If you bought Pro, you have a 14-day money-back guarantee, no questions asked.
Is switching power modes safe? I keep mine on High performance just in case.
Yes. Power modes are a native Windows feature — every laptop switches them dozens of times a day as it's plugged in and unplugged. PowerDoze writes settings only through Windows' official powercfg interface (CPU limit, cooling policy, sleep and screen timeouts) and never touches the BIOS, drivers, fan curves or your overclock. Uninstall it and everything returns to Windows defaults. Staying on High performance 24/7 doesn't add safety — it just burns extra watts while the PC waits for you.
Isn't this already built into Windows 11?
Windows 11's Presence Sensing only runs on laptops that have a hardware presence sensor, and it only locks or dims the screen — it doesn't switch power modes, combine time / app / Wi-Fi / activity signals, or show you a savings report. PowerDoze detects presence in software (window focus + input activity), so it works on any desktop or laptop and ties presence into a complete power strategy with real savings tracking.
