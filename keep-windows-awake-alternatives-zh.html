讓 Windows 不休眠

讓 Windows 該醒時醒、該睡時睡

誠實看 Caffeine、PowerToys Awake、Don't Sleep、StayGreen —— 以及它們全都仰賴「你記得去關」的那個共同缺口。PowerDoze 就是來補這個洞的:讀四個訊號、替你決定,然後讓開。

你搜的是「怎麼讓 Windows 不睡」,下面每個工具都做得到。真正的問題是下一個:誰把它關回去?Caffeine、PowerToys Awake、Don't Sleep 會一直撐著,直到你想起來把開關關掉 —— 忘了,你的電腦就整夜不睡、白白燒電。這個洞,正是 PowerDoze 存在的理由:它只在需要的 App 跑著時讓螢幕保持清醒,跑完就自己放手 —— 而且對電源模式、Wi-Fi 位置、電池也做同一招,還把你省下的電算給你看。

逐一誠實點評

Caffeine(Zhorn Software)

極輕量的經典:一支約 300KB 的免安裝程式,每 59 秒模擬按一下一個你鍵盤上根本沒有的按鍵(F15),讓 Windows 以為你還在。免費、免系統管理員。它的命令列參數其實意外強大 —— -activeperiods 指定時間窗、-watchwindow 只在某視窗開著時啟動 —— 但全是命令列、每次啟動帶參數、沒有 GUI。它也不保持螢幕開著(只防系統睡),而且預設的 F15 會干擾部分程式。

適合:想要最輕、又肯用命令列開關的人。

PowerToys Awake(微軟)

官方、免費、開源(MIT),內建於 PowerToys。模式乾淨:無限期清醒、維持一段時間、或到某個日期時間;可選「保持螢幕開」,還有 --pid 旗標讓它在某程序結束時自動退場。它唯一的罩門:鎖定畫面下完全失效。微軟自己的文件這樣寫,並叫你那種情況直接改 Windows 電源計畫。沒有每日排程、沒有 Wi-Fi、不切電源模式、沒有報表。

適合:偶爾在沒鎖屏的情況下擋一下睡眠。老實說 —— 如果你只需要這個,用 PowerToys Awake 就好,免費又官方,你不需要我們。

Don't Sleep(SoftwareOK)

防睡工具裡功能最多的一個:擋關機、待命、休眠、登出與螢幕保護,有計時器與條件 —— 電池低於 X%、CPU 或網路負載低時、或到指定時間才解除。免費、免安裝、40+ 語言。但它本質仍是「進階手動擋睡 + 條件放它睡」,不是依情境自動化 —— 而且不改你的電源計畫、什麼都不報。UI 偏密集、工程味重。

適合:想要細緻控制「何時才可以睡」的進階使用者。

StayGreen(Automata Labs)

這裡最聰明的單訊號工具:一支 Microsoft Store 應用(約 US$3–5 一次買斷、無訂閱),只在你指定的 App 位於前景時保持清醒,拔充電/關蓋自動暫停,還加了真正的熱防護(自動暫停,避免筆電在包包裡過熱),用事件驅動的 hook、0% CPU。該給的肯定:它的 per-app 觸發與熱防護是真功能,而且PowerDoze 沒做熱防護那塊。它沒有的:時間排程、Wi-Fi 位置、切電源模式、任何省電報表。

適合:要「跑某個 App 時不睡」加上擔心筆電過熱的人。

PowerDoze 補的是哪一塊

看出模式了嗎:上面每個工具都是一個 keep-awake 開關 —— 手動,或只盯一個訊號。PowerDoze 是另一個層級 —— 多訊號電源自動化。它不只擋睡,而是依情境切換你整套電源行為、然後退場,讓你不必再當那個開關:

  • 會自己關掉的白名單 —— App 跑著時保持清醒,App 一關就自動恢復可睡。不會「忘了切回去」。
  • 四個訊號,自動 —— 前景 App、閒置/離開、Wi-Fi 位置(公司 vs 家裡設定檔)、電池狀態。
  • 切換實際的 Windows 電源模式(CPU、散熱、GPU 功耗上限)—— 不只是「別睡」。
  • 真實省電報表 —— kWh、CO₂、電費估算,含歷史圖與 CSV 匯出。
  • 離開自動調暗/鎖屏,用軟體 presence 訊號 —— 任何機器都能用,不必硬體感測器。
  • 全部本機執行、免帳號、免費可用;情境加值功能是 Pro 一次買斷,無訂閱。

PowerDoze 不做什麼

因為這頁的重點就是誠實:

  • 不做電池充電上限(80% capping)。那需要韌體層控制 —— 去你筆電廠商的工具(Lenovo Vantage、MyASUS、Dell Power Manager)。任何宣稱在 Windows 通用做到的,都該懷疑。
  • 不做硬體熱防護自動暫停。如果你的需求是「筆電在包包裡過熱就自動停」,StayGreen 那塊做得比較專。
  • 它不是 CPU 排程器。背景程序害你前景卡頓,那是 Process Lasso 的活,不是我們。

常見問題

為什麼 PowerToys Awake 一鎖屏就失效?PowerDoze 不一樣嗎?

鎖定畫面跑在獨立的安全 session,一般 user-mode 應用無法在那裡維持電源請求 —— 微軟有文件說明,並建議改用電源計畫。PowerDoze 的背景引擎在鎖屏下照樣執行,所以它的規則仍然生效。

Caffeine、Don't Sleep 都免費,我為什麼要付費給 PowerDoze?

PowerDoze 也免費可用。只有情境加值(Wi-Fi 位置、會議偵測、離開調暗/鎖屏、省電分析)是 Pro 一次買斷、無訂閱。如果你只需要一個手動的 keep-awake 開關,那些免費工具很好;我們是給「受夠了當那個手動切換的人」用的。

我只是想下載大檔時電腦別睡,該用哪個?

老實說,用 PowerToys Awake 或 Caffeine。它們免費、又剛好做這個。PowerDoze 的價值在於「讓它自動依情境決定、切換電源行為」,不是一個手動開關。

PowerDoze 會不會像它們一樣讓電腦永遠不睡?

只在觸發持續時。它的 App 白名單在該 App 跑著時保持清醒,App 一關就自動恢復正常睡眠 —— 這正是重點。

不想用 mouse jiggler,最簡單怎麼讓 Windows 不休眠?

這些工具都比實體甩鼠器乾淨 —— 它們是對 Windows 宣告一個真正的電源請求,而不是假裝有輸入。PowerDoze(或手動情況用 PowerToys Awake)做這件事不會有幻影滑鼠移動。

結論

只想要一個手動開關?PowerToys Awake。最輕、肯用命令列?Caffeine。要細控「何時可以睡」?Don't Sleep。要 per-app + 熱防護?StayGreen —— 它們各自把那一件事做得很好。但如果你受夠了當那個「要記得把電源設定切來切去」的人,這裡只有 PowerDoze 是為了替你做這件事而生的 —— 自動、看四個訊號,還把省下的算給你看。
