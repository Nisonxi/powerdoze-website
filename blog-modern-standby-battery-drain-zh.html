筆電「睡著」還在偷跑電？元兇是 Modern Standby

Nisonxi

早上把筆電一闔，塞進包包就出門。中午打開——咦，怎麼燙燙的？電還掉了一截，你明明整個上午都沒碰它。

它沒壞。真相比「壞掉」更微妙：你以為它在睡覺，它其實一直半醒著偷上網，邊發燙邊吃你的電。

別擔心，這篇就帶你把這個躲在「睡眠」底下的小偷揪出來——而且大多數人，三個步驟就能搞定。

說好的「睡覺」，被偷偷換掉了

我們都以為「螢幕一闔 = 電腦睡了 = 不耗電」。以前真的是這樣。

過去二十年，筆電闔上叫做 S3 睡眠：只留記憶體一點點電、其他全斷掉，整台幾乎不吃電，放三五天再開都還有電。但這幾年的新筆電悄悄換了規則，現在的睡眠叫 Modern Standby（S0 低功耗待機），理念跟手機一樣——所謂「睡著」其實是淺淺地醒著：隨時連網、幫你同步信件、偷偷裝更新，你一掀蓋它就秒醒。

順利時很方便。但只要有某個程式或驅動「卡」住、讓它一直高耗電，你的包包就變成保溫箱了。健康的機器一整晚只該掉幾趴；出狀況的，一小時能燒掉 10% 以上——名義上在睡，其實累得半死。

傳統睡眠像睡得很沉的人：你不搖他，他基本上跟關機沒兩樣。Modern Standby 則像淺眠的人，手機放在床頭——一震動、一則通知就半醒過來瞄一眼。多數晚上沒事，但要是手機整夜亮個不停，他根本沒真的休息到，隔天又累又燙。那，就是躺在你包包裡的筆電。

第一步：先問問電腦它在搞什麼

別急著亂改設定，先讓 Windows 自己招供。開「系統管理員」終端機（開始選單搜尋 cmd → 右鍵「以系統管理員身分執行」），打三個指令：

  • powercfg /a — 看你的筆電支援哪幾種睡眠。看到「待命 (S0 低電源閒置)」就是中了 Modern Standby；如果清單裡根本沒有 S3，代表硬體就不給你老式睡眠，任何軟體都救不回（這點等下有用）。
  • powercfg /sleepstudy — 這個最好用。它生出一份網頁報告，把你每一次「睡眠」攤開：睡多久、吃多少電、誰害它睡不著。兇手通常就寫在「Top Offenders（最大元兇）」那欄。
  • powercfg /lastwake — 想知道「上次是誰把它吵醒的？」這行告訴你。
sleepstudy 就像幫筆電做一份「睡眠品質報告」，跟智慧手環記錄你的睡眠一樣：它不會幫你睡得更好，但會明明白白告訴你——昨晚幾點被吵醒、被什麼吵醒。知道兇手，才好對症下藥。

第二步：對症下藥

抓到兇手後，通常這幾招就能解決，從最簡單的開始：

  • 把愛亂叫醒它的東西「禁聲」。跑 powercfg /devicequery wake_armed 會列出「有權叫醒電腦」的裝置，裡面常有用不到的網路卡、滑鼠鍵盤。到「裝置管理員」找到它 → 內容 → 電源管理 → 取消「允許這個裝置喚醒電腦」。
  • 睡覺時順手把網路斷掉。Modern Standby 耗電很大一塊就是一直掛在網路上。有些筆電的電源設定有「待命時的網路連線」，把（用電池時）設成「停用」，它睡起來就乖很多。
  • 要離開久一點，改用「休眠」。這招最關鍵。休眠（S4）會把記憶體存到硬碟、然後整台真正斷電——零耗電，放一整天、坐一趟飛機都沒事，只是再開機多等幾秒（離開好幾小時的話根本無感）。先確認休眠開著（powercfg /hibernate on），再把「用電池闔蓋時」的動作從睡眠改成休眠就好。
休眠就像出國前把冰箱拔插頭、清空、上鎖：不耗電、不會壞、一毛不花，代價是回家要等它重新冷起來。睡眠則只是把冰箱門關上：門一開就是冷的，但全程都在低聲運轉。喝杯咖啡的休息，關門就好；出門兩週，就該拔插頭。把筆電塞進包包出門，就是那趟兩週的旅行。

至於網路上很多人推的「改登錄檔把 Modern Standby 整個關掉」（PlatformAoAcOverride）——先別衝動。如果你的機器硬體上已經沒有 S3（剛剛 powercfg /a 沒看到的話，近幾年筆電大多如此），關掉之後你換不回老式睡眠，只會得到一個怪怪、沒人保證的狀態，而且新版 Windows 有時根本不理它。真要試請隨時準備改回來；但老實說，對大部分人「闔蓋改休眠」就有一樣效果，還不用冒這個險。

問題的根源，其實是「該用哪一種」

你有沒有發現，上面這些招會「打架」？塞進包包你要休眠（完全斷電）；放在書桌上你卻要輕快的睡眠（掀開秒醒）。同一台筆電，不同情境要的不一樣。

偏偏 Windows 只讓你設一組「闔蓋要做什麼」。所以不管選哪邊，總有一半時間是錯的：設休眠，在家掀蓋要等；設睡眠，出門又繼續偷耗電。

這就是 PowerDoze 在幫你顧的那一層。它不會去改 Modern Standby 本身（那是硬體和 Windows 的地盤，要硬碰硬請用上面的診斷三步驟）；它幫你管的是什麼時候、用哪一種方式睡——電，大多就是在這裡省回來的：

  • 闔蓋動作分插電 / 用電池兩種——「用電池就休眠、接上 dock 就睡眠」，把上面那帖「包包處方」變自動，你再也不用記。
  • 設一個半夜自動休眠——例如「凌晨一點還醒著就自動休眠」，專門接住那些「忘了關就丟著」的夜晚（這其實是早上電量莫名變少的最大宗）。
  • 睡眠逾時跟著電源模式走——電池模式抓緊、插電放寬，自動切換，不用每次再進控制台。

老實說一句

任何第三方軟體——包括我們的——都沒辦法改變 Modern Standby 進入之後的行為，那是硬體和 Windows 說了算，要硬碰硬請走上面的診斷路線。但軟體能做、而且最有價值的，是那個「決策」：每一次都讓電腦在對的時機、用對的方式進入低功耗（該睡就睡、該休眠就休眠、該關就關），而你完全不用動腦。實際上，大多數莫名消失的電，一直都漏在這個決策層裡。想把 Windows 睡眠設定一次搞懂，可以接著讀我們的 Windows 睡眠設定完整指南。
